# jQuery

## Nome
jQuery

## Descrição
Nesse repositório foi possível aprender e colocar em prática as funcionalidades do jQuery

## Funcionalidades
    - Criação de eventos
    - Filtros básicos
    - Seletores básicos
    - Eventos (mouse, teclado, formulário, browser, personalizado)

## Tecnologias
- JavaScript
    -jQuery v3.6
- Node / npm
  
## Inicialização
Executar no terminal os comandos: 
``` terminal
npm install --global yarn
npm install
```
